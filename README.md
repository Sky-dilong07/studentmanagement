# Student Management System

A simple **Student Management System** built using **Spring Boot**, **MySQL**, and a **HTML/JS frontend**.  
This project allows you to **Add, Update, Delete, and View students** with a clean and responsive interface.

---

## Features

- Add a new student (Name, Email, Course)
- Update existing student details
- Delete students
- View all students in a table
- Fully integrated **backend (Spring Boot)** and **frontend (HTML + JS)**

---

## Technologies Used

- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** IntelliJ IDEA, GitHub

---

## Getting Started

### Prerequisites

- Java JDK 21 or above
- MySQL Database
- Maven
- IntelliJ IDEA or any Java IDE

### Steps to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-management.git
cd student-management

2. Create a MySQL database named student_db and update application.properties with your credentials:
spring.datasource.url=jdbc:mysql://localhost:3306/student_db
spring.datasource.username=root
spring.datasource.password=YOUR_PASSWORD

3. Run the Spring Boot application (backend).

4. Access the frontend in your browser:
http://localhost:9090/index.html
